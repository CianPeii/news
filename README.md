### News Website
一個使用 React 和 Vite 建立的新聞網站專案，提供直覺的操作界面與全面的閱讀體驗。使用者可以瀏覽新聞、切換深色模式、選擇語言，並透過直覺的界面快速瀏覽感興趣的主題。

##### 使用技術
React 18.3.1 - 前端框架
Vite 6.0.1 - 開發伺服器與建置工具
Tailwind CSS 3.4.16 - CSS 框架
React Router 7.0.2 - 路由管理
Axios 1.7.9 - API 請求工具
React i18n -多國語言切換

##### 主要功能
搜尋功能
新聞瀏覽
新聞分類
多國語系
深色模式
RWD 響應式網站

##### 📦 安裝與執行

1️⃣ 安裝依賴

npm install

2️⃣ 啟動開發伺服器

npm run dev

3️⃣ 打包專案

npm run build
